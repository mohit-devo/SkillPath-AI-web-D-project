# 📖 Overview

**SkillPath AI** is an AI-powered full-stack web application that helps learners create personalized learning journeys based on their goals, interests, and current skill levels.

The platform leverages Artificial Intelligence to generate customized learning roadmaps, recommend real-world projects, provide an intelligent learning assistant for doubt solving, and track user progress through an interactive dashboard.

Built using the **MERN Stack**, SkillPath AI aims to make learning more personalized, accessible, and engaging while supporting **United Nations Sustainable Development Goal 4 (Quality Education).**

# 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/mohit-devo/SkillPath-AI-web-D-project

cd SkillPath-AI
```

### Install Frontend

```bash
cd client
npm install
```

### Install Backend

```bash
cd ../server
npm install
```

---

# ▶️ Run the Project

### Start Backend

```bash
cd server
npm run dev
```

### Start Frontend

```bash
cd client
npm run dev
```

# 📡 API Endpoints

| Method | Endpoint             | Description                                |
| ------ | -------------------- | ------------------------------------------ |
| POST   | `/api/auth/register` | Register a new user                        |
| POST   | `/api/auth/login`    | Authenticate user                          |
| GET    | `/api/user/profile`  | Retrieve user profile                      |
| POST   | `/api/roadmap`       | Generate AI learning roadmap               |
| POST   | `/api/chat`          | AI-powered learning assistant              |
| GET    | `/api/projects`      | Fetch personalized project recommendations |
